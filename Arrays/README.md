# 📂 01 - Arrays

This directory contains my Java solutions for Array-based LeetCode problems. The problems are structured progressively by difficulty to build a strong foundation in array manipulation, traversal, and optimization techniques (such as Two Pointers and Sliding Window).

## 🎯 Learning Objectives
* Master in-place array modifications.
* Understand the time and space complexity trade-offs of different array operations.
* Transition from brute-force iterations to optimized $O(N)$ and $O(\log N)$ algorithms.
